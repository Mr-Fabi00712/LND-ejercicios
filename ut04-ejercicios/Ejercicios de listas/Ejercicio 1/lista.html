Terminología Web

HTTP
El Protocolo de Transferencia de Hipertexto (HTTP) es el protocolo de comunicación que permite las transferencias de información en la World Wide Web.
HTML
HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para la elaboración de páginas web.
URL
Un Localizador de Recursos Uniforme (URL) es una secuencia de caracteres que identifica recursos en una red, como la World Wide Web.
TCP/IP
El Protocolo de Control de Transmisión/Protocolo de Internet (TCP/IP) es un conjunto de protocolos de red que permite la comunicación entre computadoras en una red.
Internet
Internet es una red global de computadoras interconectadas que permite la comunicación y el intercambio de información en todo el mundo.
W3C
El World Wide Web Consortium (W3C) es una organización internacional que desarrolla estándares para la web, garantizando su crecimiento y accesibilidad.
Mozilla Foundation
La Mozilla Foundation es una organización sin fines de lucro que promueve la apertura, innovación y oportunidad en la web, siendo conocida por desarrollar el navegador Firefox.
