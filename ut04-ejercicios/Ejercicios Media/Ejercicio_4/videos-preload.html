Video con preload: metadata

El atributo preload tiene el valor metadata, lo que significa que solo se descargará la información del video (como la duración y los metadatos), pero no se descargará el video completo antes de que se inicie la reproducción.

Otros valores posibles de preload:

  • auto: El navegador intentará cargar el video completo tan pronto como sea posible, incluso antes de que el usuario haga clic en el botón de reproducción. Esto asegura que el video esté listo para reproducirse de inmediato.
  • none: El navegador no pre-cargará nada. Solo se descargará el video cuando el usuario lo reproduzca, lo que ahorra ancho de banda pero puede causar un pequeño retraso cuando se inicia la reproducción.

Comportamiento de cada valor:

preload="auto": El video se descargará completamente tan pronto como la página se cargue. Esto es útil si quieres asegurar que el video esté completamente disponible para reproducirse sin interrupciones.

preload="metadata": Solo se descargan los metadatos (como duración, dimensiones) para mostrar una vista previa y preparar el video, pero no se descarga el contenido completo del video.

preload="none": El video no se descargará de inmediato. Solo se descargará cuando el usuario presione el botón de reproducción, lo que es ideal si no quieres que el video use ancho de banda innecesario hasta que sea necesario.
